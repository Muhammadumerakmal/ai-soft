# Project Charter

## Vision
To create an intuitive task management app that enhances team collaboration and productivity through efficient project tracking.

## Scope
The project will include features for creating projects, adding tasks with due dates, assigning tasks to team members, and tracking task completion on a kanban board. It will not include advanced features like time tracking, integrations with other tools, or mobile app versions in the initial release.

## Target Audience
- Project Managers
- Team Leaders
- Remote Teams
- Freelancers

## Success Criteria
- User adoption rate of at least 500 active users within the first three months
- User satisfaction score of 80% or higher in feedback surveys
- Completion of tasks tracked on the kanban board should improve by 30% within the first six months of use

## Recommended Tech Stack
- next.js
- express
- postgresql
